// turbo-all

1. Open in Android Studio (optional)
   `open -a "Android Studio" .`

2. Build and run via Gradle
   `./gradlew :compose-app:installDebug`

   > [!TIP]
   > Ensure you have an Android emulator running or device connected: `adb devices`
